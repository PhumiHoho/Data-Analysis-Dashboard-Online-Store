# 🛍️ Online Clothing Store Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** developed for an **Online Clothing Store** that sells apparel for both **Men** and **Women**. The dashboard provides valuable business insights into sales performance, customer behaviour, inventory levels, and order management to support data-driven decision-making.

The report was created using Microsoft Power BI and demonstrates data modelling, DAX calculations, relationships, KPI design, and interactive visualisations.
